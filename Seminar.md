## 02

```
START
  SET target_number TO 1
  SET kast_count TO 0
  WHILE target_number <= 6
    ROLL tärning
    INCREMENT kast_count
    IF resultat från tärning EQUALS target_number
      INCREMENT target_number
    END IF
  END WHILE
  PRINT kast_count
END
```
